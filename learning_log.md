## Week 3: Data Cleaning with Pandas

### 📌 Topics Covered
- Handling Missing Values
- Detecting Missing Data using `isnull()`
- Removing Missing Values using `dropna()`
- Filling Missing Values using `fillna()`
- Forward Fill using `ffill()`
- Backward Fill using `bfill()`
- Handling Duplicate Data
- Identifying and Managing Duplicate Rows

### 🧠 Key Takeaways
- Real-world datasets often contain missing and duplicate values.
- `isnull()` helps identify missing data in a DataFrame.
- `dropna()` removes rows or columns containing missing values.
- `fillna()` can replace missing values with meaningful data.
- `ffill()` and `bfill()` propagate nearby values to fill missing entries.
- Duplicate records can affect analysis and lead to inaccurate results.
- Data cleaning is an essential step before performing data analysis or building machine learning models.
